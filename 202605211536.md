### 2. 把 HOG 特征强制转 float32

在 joint 路径中，HOG 提取完成后立即做：

zscore 也建议输出 float32：

```python
def zscore_float32(X, eps=1e-6):
    X = np.asarray(X, dtype=np.float32, order="C")

    mu = X.mean(axis=0, dtype=np.float64)
    std = X.std(axis=0, dtype=np.float64)

    bad = (~np.isfinite(std)) | (std < eps)
    std[bad] = 1.0

    X = (X - mu.astype(np.float32)) / std.astype(np.float32)
    X[~np.isfinite(X)] = 0.0

    return np.ascontiguousarray(X, dtype=np.float32)
```

### 3. 替换 pairwise_euclidean 为 chunked GEMM 版本

不要用三维广播。建议在 `inv_common.py` 或 `inv_joint.py` 内临时加安全版本：

```python
def pairwise_euclidean_chunked(X, Y=None, batch_size=256):
    import numpy as np

    X = np.asarray(X, dtype=np.float32, order="C")
    if Y is None:
        Y = X
        same = True
    else:
        Y = np.asarray(Y, dtype=np.float32, order="C")
        same = False

    n = X.shape[0]
    m = Y.shape[0]

    YT = np.ascontiguousarray(Y.T, dtype=np.float32)

    x2 = np.sum(X * X, axis=1, dtype=np.float64).astype(np.float32)
    y2 = np.sum(Y * Y, axis=1, dtype=np.float64).astype(np.float32)

    D = np.empty((n, m), dtype=np.float32)

    for i in range(0, n, batch_size):
        j = min(i + batch_size, n)

        D2 = -2.0 * (X[i:j] @ YT)
        D2 += x2[i:j, None]
        D2 += y2[None, :]

        np.maximum(D2, 0.0, out=D2)
        np.sqrt(D2, out=D2)

        D[i:j] = D2

        print(f"[pairwise] rows {i}:{j} done", flush=True)

    if same:
        np.fill_diagonal(D, np.inf)

    return D
```

这个版本内存大约是：

```text
X_img float32      ≈ 87 MB
D_img float32      ≈ 29 MB
YT float32         ≈ 87 MB
chunk 临时矩阵     < 5 MB
总计远小于 1 GB
```

可以先用 `batch_size=128`，确认稳定后再改 256 或 512。
