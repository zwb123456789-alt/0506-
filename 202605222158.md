## 本轮任务

新增脚本：

```text
ocs_project/03_inversion/fuse_predictions.py
```

功能：

```text
读取已训练好的 OCS-only MLP 预测结果
读取已训练好的 CNN image-only 预测结果
按 yaw_true / pitch_true 对齐 test 样本
在 sin/cos 空间做预测级融合
sweep beta 得到最佳 OCS 权重
```

融合定义：

```text
y_fused = beta * y_ocs + (1 - beta) * y_img
```

其中：

```text
beta = OCS prediction 权重
beta = 0.0  -> CNN image-only
beta = 1.0  -> OCS-only MLP
```

注意：必须在 sin/cos 空间融合，不要直接平均角度。

## 脚本实现要求

### 1. 自动扫描预测文件

`mlp_ocs/run_20260521_084723/` 里可能包含多个特征、多 seed 的预测 CSV。  
脚本需要先打印候选文件，例如：

```python
for p in run_dir.glob("*pred*.csv"):
    print(p.name)
```

如果文件名规则不统一，就支持手动 glob：

```text
--ocs-glob "*all*raw*seed*.csv"
--cnn-glob "predictions_seed*.csv"
```

### 2. 每个 seed 单独融合

对 seed 0~4 分别做：

```text
OCS seed i prediction
CNN seed i prediction
align by yaw_true / pitch_true
beta sweep
```

### 3. 对齐方式

用真实姿态对齐：

```text
key = (round(yaw_true, 6), round(pitch_true, 6))
```

必须检查：

```text
N_ocs_test = 1998
N_cnn_test = 1998
N_matched = 1998
```

否则暂停排查。

### 4. sin/cos 融合

从预测角度还原 sin/cos：

```python
def angle_to_vec(yaw_deg, pitch_deg):
    y = np.deg2rad(yaw_deg)
    p = np.deg2rad(pitch_deg)
    return np.stack([
        np.sin(y), np.cos(y),
        np.sin(p), np.cos(p),
    ], axis=1)
```

融合后 decode：

```python
vec = beta * vec_ocs + (1.0 - beta) * vec_img

yaw = np.rad2deg(np.arctan2(vec[:, 0], vec[:, 1])) % 360.0
pitch = np.rad2deg(np.arctan2(vec[:, 2], vec[:, 3]))
```

### 5. beta sweep

先做：

```text
beta = 0.00, 0.01, ..., 1.00
```

输出每个 beta 的 5 seeds 平均：

```text
mean
std
median
p90
p95
Hit5
Hit10
yaw_mean
pitch_mean
```

### 6. endpoint sanity check

必须满足：

```text
beta = 0.00 约等于 CNN image-only 结果
beta = 1.00 约等于对应 OCS MLP 结果
```

允许极小浮点差异。
