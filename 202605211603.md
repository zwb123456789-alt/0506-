## 代码任务

新增：

```text
ocs_project/03_inversion/train_cnn.py
```

要求：

1. 复用 `inv_common.py` 中已有的数据读取、角距、metrics、保存工具。
2. 图像路径兼容当前 GGX PNG：

```text
out_prefix -> *_brdf.png
```

3. 输入图像使用 phase63 的 2701 张 PNG。
4. 标签为 yaw / pitch。
5. 输出编码沿用 `train_mlp.py`：

```text
[sin(yaw), cos(yaw), sin(pitch), cos(pitch)]
```

6. 预测时归一化 sin/cos 后 decode 回 yaw / pitch。
7. split 复用 `train_mlp.py` 的 10°→5° 逻辑：
   - 10° 网格作为 train pool
   - 5° 非训练网格作为 test
   - train pool 内部再按 seed 做 train / val
8. Windows 下 DataLoader 必须默认：

```text
num_workers = 0
```

9. 模型先用小 CNN，不要上复杂大模型：

```text
Conv/GN/SiLU/Pool × 4
AdaptiveAvgPool
MLP head -> 4
```

建议结构：

```python
1ch image
Conv 1->16
Conv 16->32
Conv 32->64
Conv 64->128
AdaptiveAvgPool2d(1)
Linear 128->64
Linear 64->4
```

10. 图像预处理至少支持两个模式：

```text
--intensity raw
--intensity log1p
```

其中 `log1p` 可用：

```python
x = log1p(10 * x) / log1p(10)
```

11. 输出指标：

```text
mean / median / p90 / p95
Top1@5°
Top1@10°
yaw_mean
pitch_mean
```

12. 每个 seed 保存：

```text
predictions_seed*.csv
metrics_seed*.json
best_model_seed*.pt
```

13. 总结保存：

```text
summary.json
summary.csv
config_used.json
```
